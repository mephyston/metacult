# Component Inventory

**Library:** `@metacult/shared-ui`
**Framework:** Vue 3
**Styling:** Tailwind CSS

## Feature Components

High-level business components.

- **Hero**: Landing page hero sections (`src/components/features/hero`).
- **Review**: Review management UI (`src/components/features/review`).
  - `ReviewCard.vue`
  - `ReviewDeck.vue`
- **Swipe**: Tinder-style card interface (`src/components/features/swipe`).
  - `SwipeCard.vue`
  - `SwipeDeck.vue`

## UI Primitives

Reusable, atomic UI elements (Design System).

| Component    | Path               | Description                                   |
| ------------ | ------------------ | --------------------------------------------- |
| **Avatar**   | `ui/avatar`        | User profile images with fallbacks            |
| **Badge**    | `ui/badge`         | Status indicators and tags                    |
| **Button**   | `ui/button`        | Interactive buttons                           |
| **Card**     | `ui/card`          | Content containers with header/content/footer |
| **Carousel** | `ui/carousel`      | Image/Content sliders                         |
| **Command**  | `ui/command`       | Command palette interface                     |
| **Dialog**   | `ui/dialog`        | Modal windows                                 |
| **Dropdown** | `ui/dropdown-menu` | Menu overlays                                 |
| **Heading**  | `ui/heading`       | Typography component                          |
| **Input**    | `ui/input`         | Text input fields                             |
| **Label**    | `ui/label`         | Form labels                                   |

## Layout Components

- **Header**: Global site header.
- **Footer**: Global site footer.
- **Navigation**: Main navigation menu.
- **Search**: Global search interface.
